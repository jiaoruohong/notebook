## Compile

- Compile in English: </br>
  ```bash
  latexmk -pdflatex=pdflatex -pdf ./note-en.tex
  ```
- Compile in Chinese: </br>
  ```bash
  latexmk -pdflatex=xelatex -pdf ./note-cn.tex
  ```
